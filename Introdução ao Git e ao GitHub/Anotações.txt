O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos Básicos:
mkdir " " - Cria uma pasta.
trl+l - Limpa o terminal.
dir - Entrar em uma pasta.
ls - exibir o conteudo da pasta atual.
cd .. - volta uma pasta anterior.
git add . - Atualiza a lista de arquivos no Git para fazer o commit no GitHub.
